OpenClaw - 开源 AI 助手平台

让 AI 助手融入你的聊天应用

OpenClaw 是一个开源的 AI 助手框架，可以无缝集成到各种消息平台中，包括 Telegram、WhatsApp、Feishu、Discord 等。

开始安装 快速入门

主要特性

多平台支持

支持 Telegram、WhatsApp、Feishu、Discord、Slack 等主流消息平台

本地运行

完全在本地运行，保护你的隐私和数据安全

强大的命令系统

丰富的命令行工具，支持文件操作、浏览器控制、设备管理等

可扩展架构

通过插件和技能系统轻松扩展功能

快速开始

# 安装 OpenClaw
npm install -g openclaw

# 初始化配置
openclaw setup

# 启动服务
openclaw gateway start

详细安装和配置指南请查看 安装指南 页面。

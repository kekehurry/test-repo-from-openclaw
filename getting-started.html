入门指南

快速开始使用 OpenClaw，从安装到第一个消息发送。

快速开始

OpenClaw 是一个强大的 AI 助手框架，可以让你的 AI 直接在各种消息平台中工作。以下是快速开始的步骤：

1. 安装 OpenClaw

首先确保你已经安装了 Node.js (v18 或更高版本)，然后运行：

# 全局安装 OpenClaw
npm install -g openclaw

# 或使用 yarn
yarn global add openclaw

2. 初始化配置

运行设置向导来配置你的消息渠道：

openclaw setup

这个向导会引导你配置 Telegram、WhatsApp、Feishu、Discord 等消息渠道。

3. 启动 Gateway 服务

Gateway 是 OpenClaw 的核心服务，负责处理所有消息和命令：

openclaw gateway start

4. 发送第一个消息

现在你可以通过 OpenClaw 发送消息了：

# 发送到 Telegram
openclaw message send --channel telegram --target @yourusername --message "Hello from OpenClaw!"

# 发送到 WhatsApp
openclaw message send --channel whatsapp --target +1234567890 --message "Hello from OpenClaw!"

实际示例

以下是一个完整的流程示例：

# 1. 安装
npm install -g openclaw

# 2. 配置（按照向导提示）
openclaw setup

# 3. 启动服务
openclaw gateway start

# 4. 检查状态
openclaw status

# 5. 发送测试消息
openclaw message send --target your-chat-id --message "OpenClaw is working!"

基本概念

在深入使用之前，了解一些基本概念很重要：

Gateway

Gateway 是 OpenClaw 的核心服务，它作为一个 WebSocket 服务器运行，处理所有的消息路由、命令执行和状态管理。

Agent

Agent 是实际的 AI 助手，它接收消息、处理请求并生成回复。你可以有多个不同的 Agent 配置。

Channels

Channels 是消息平台的连接器，如 Telegram、WhatsApp、Feishu 等。每个 Channel 负责与特定平台的通信。

Skills

Skills 是扩展功能模块，为 Agent 提供特定的能力，如文件操作、网络搜索、系统控制等。

重要提示

首次使用时，请确保你的系统满足以下要求：

  • Node.js v18 或更高版本
  • 至少 2GB 可用内存
  • 稳定的网络连接（用于消息平台认证）
  • 适当的防火墙设置（允许 WebSocket 连接）

安装指南

详细说明如何在不同操作系统上安装和配置 OpenClaw

安装前提条件

在安装 OpenClaw 之前，请确保您的系统满足以下要求：

  • Node.js: v18 或更高版本
  • npm 或 yarn: 包管理器
  • 支持的操作系统: Linux、macOS、Windows (WSL)
  • 网络连接: 用于下载依赖包和连接消息渠道

重要提示

OpenClaw 需要至少 2GB 内存才能正常运行。如果您的系统内存不足，请考虑关闭其他应用程序或升级系统内存。

Linux / macOS 安装

在 Linux 或 macOS 系统上，您可以使用 npm 或 yarn 进行全局安装：

使用 npm 安装

# 全局安装 OpenClaw
npm install -g openclaw

# 初始化配置
openclaw setup

使用 yarn 安装

# 全局安装 OpenClaw
yarn global add openclaw

# 初始化配置
openclaw setup

权限问题解决

如果您遇到权限错误，可以尝试以下解决方案：

# 方案1: 使用 npx（无需全局安装）
npx openclaw setup

# 方案2: 修复 npm 全局目录权限
sudo chown -R $(whoami) $(npm config get prefix)/{lib/node_modules,bin,share}

# 方方案3: 使用 Node 版本管理器（推荐）
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install --lts
npm install -g openclaw

Windows (WSL) 安装

在 Windows 系统上，建议使用 WSL (Windows Subsystem for Linux) 进行安装：

安装 WSL 和 Node.js

# 在 PowerShell 中以管理员身份运行
wsl --install

# 重启后，在 WSL 中安装 Node.js
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get install -y nodejs

# 验证安装
node --version
npm --version

安装 OpenClaw

# 安装 OpenClaw
npm install -g openclaw

# 初始化配置
openclaw setup

配置向导

运行 openclaw setup 后，您将进入交互式配置向导：

  1. 选择消息渠道: Telegram、WhatsApp、Feishu、Discord 等
  2. 提供 API 凭证: 根据选择的渠道提供相应的认证信息
  3. 配置 Gateway: 设置 WebSocket 服务端口和安全选项
  4. 测试连接: 验证配置是否正确

常见渠道配置要求

  • Telegram: 需要 Bot Token（通过 @BotFather 获取）
  • WhatsApp: 需要扫描 QR 码连接（个人账号）
  • Feishu: 需要企业应用凭证（App ID 和 App Secret）
  • Discord: 需要 Bot Token（在 Discord Developer Portal 创建）

验证安装

安装完成后，可以通过以下命令验证 OpenClaw 是否正常工作：

# 查看版本信息
openclaw --version

# 查看帮助信息
openclaw --help

# 启动 Gateway 服务
openclaw gateway start

# 查看系统状态
openclaw status

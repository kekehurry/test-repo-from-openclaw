skills 命令

发现、扫描和配置 OpenClaw 技能（Skills）

命令概述

skills 命令用于管理和操作 OpenClaw 的技能系统。技能是 OpenClaw 的核心扩展机制，允许 AI 助手执行特定任务，如文件操作、网络搜索、设备控制等。

技能类型

  • 内置技能: 随 OpenClaw 一起提供的核心技能
  • 社区技能: 来自 ClawHub 社区的第三方技能
  • 自定义技能: 用户自己开发的专用技能

子命令列表

  • skills list - 列出所有可用技能
  • skills info <skill-name> - 显示技能详细信息
  • skills install <skill-url> - 安装新技能
  • skills uninstall <skill-name> - 卸载技能
  • skills update <skill-name> - 更新技能
  • skills enable <skill-name> - 启用技能
  • skills disable <skill-name> - 禁用技能
  • skills create - 创建新技能模板

技能管理

列出技能

# 列出所有已安装的技能
openclaw skills list

# 列出启用的技能
openclaw skills list --enabled

# 列出禁用的技能  
openclaw skills list --disabled

# 详细列表（包含版本信息）
openclaw skills list --verbose

查看技能信息

# 查看 weather 技能的详细信息
openclaw skills info weather

# 查看技能的依赖关系
openclaw skills info weather --dependencies

# 查看技能的配置选项
openclaw skills info weather --config

安装和更新技能

从 ClawHub 安装

# 从 ClawHub 安装技能
openclaw skills install weather

# 从 GitHub 安装技能
openclaw skills install https://github.com/user/weather-skill

# 从本地文件安装
openclaw skills install ./my-custom-skill

更新技能

# 更新单个技能
openclaw skills update weather

# 更新所有技能
openclaw skills update --all

# 强制重新安装
openclaw skills install weather --force

技能启用/禁用

技能可以被启用或禁用，而不必完全卸载：

# 禁用 weather 技能
openclaw skills disable weather

# 启用 weather 技能
openclaw skills enable weather

# 临时禁用所有技能（调试用）
openclaw skills disable --all

# 重新启用所有技能
openclaw skills enable --all

注意

禁用技能后，AI 助手将无法使用该技能的功能，但技能文件仍然保留在系统中。

创建自定义技能

使用 skills create 命令可以创建新的技能模板：

# 创建新的技能模板
openclaw skills create my-new-skill

# 指定技能类型
openclaw skills create my-web-skill --type web

# 在指定目录创建
openclaw skills create my-skill --path ./skills

技能结构

新创建的技能目录包含以下文件：

  • SKILL.md - 技能描述和使用说明
  • skill.json - 技能元数据和配置
  • index.js - 技能主逻辑文件
  • README.md - 详细的文档

常用技能示例

Weather 技能

# 查询天气
用户: "今天北京天气如何？"
AI: (调用 weather 技能) → "今天北京晴天，气温 25°C"

File 技能

# 文件操作
用户: "读取 ~/notes.txt 文件"
AI: (调用 file 技能) → 返回文件内容

Web Search 技能

# 网络搜索
用户: "最新的 AI 新闻"
AI: (调用 web-search 技能) → 返回相关新闻摘要

故障排除

常见问题

  • 技能未找到: 确保技能已正确安装并启用
  • 技能权限错误: 检查技能所需的系统权限
  • 技能依赖缺失: 安装技能所需的依赖包
  • 技能配置错误: 检查技能的配置文件是否正确

调试命令

# 查看技能日志
openclaw logs --skill weather

# 验证技能完整性
openclaw skills verify weather

# 重置技能配置
openclaw skills reset weather
